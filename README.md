# ML4FF Framework

This repository will contain the framework code and dataset used in the paper “ML4FF: A machine-learning framework for flash flood forecasting applied to a Brazilian watershed” by Jaqueline A. J. P. Soares, Luan C. S. M. Ozelim, Luiz Bacelar, Dimas B. Ribeiro, Stephan Stephany, and Leonardo B. L. Santos.

**The framework code and dataset will be available as soon as the article is published.**
